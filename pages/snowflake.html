Happy Holidays!

This was one of the first projects that I designed last year in A.P. Computer Science A. It was a holiday card that I sent to my brother who lives out on the East Coast. This was very challenging in that there are multiple components to it. First, you will see the layout of everything. Putting the fireplace, tree, and gifts was a difficult process. Everything needed to be placed in front of the wall, but the wall did not like it when this occurred.

Next, you will see the snowflakes. This was the hardest part –– making snowflakes randomly appear and vary in size. This took the longest to do in this project because it was very touchy in how the code was formatted.

Finally, you see the message that is changing color. This was a relatively simple task as it relied on a framecount on when to change the color of the message.

Want to see the code? Here's the link: Happy Holidays Code

I hope you enjoy my holiday card!
