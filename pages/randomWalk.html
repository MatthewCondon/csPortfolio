Put Out The Fire!

This is another project we made in Advanced computer Science. The essence of this project can be found in the various arrays and random walks used in it.

We recently learned about random walks –– the random path an object in an array will take. So, we were tasked with creating an object and allowing it to take a random walk. For the project, I designed a fire extinguisher. The random walks and arrays can be seen in two spots: the carbon bubbles and the smoke of the fire.

The hardest part of this project rested primarily with all the components. It became an early necessity to stay organized since there were multiple classes, various functions, and a lot of variables. This caused some struggles in creating the scenes because certain things needed to occur at different times but using the same function. I had to use various booleans or functions (mousePressed vs mouseIsPressed) to combat this issue.

Want to see the code? Here's the link: Random Walks and Arrays Project Code

To use the extinguisher, move it closer to the fire and hold down on the mouse. Now, quickly put out the fire before it gets too large!
