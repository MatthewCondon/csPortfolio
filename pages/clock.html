Time to wake up!

This is the alarm clock that I made for my final project last year. If you take a look at it, there are multiple functions to it. First, it constantly updates with real date and time. Next, the color scheme changes between day and night. After that, there is an alarm function (There is typically a sound function, but it kept causing the portfolio to crash. So, when the alarm was set, it would scream "Good Morning!"). Finally, you are able to set the weather by clicking in the bottom right corner.

The most challenging part about creating the clock was inputting a working sound system for the alarm. The sound file had to be a specific type and size. It was also hard creating a function that used it without altering the sound. I hope you enjoy looking at last year's capstone project!
