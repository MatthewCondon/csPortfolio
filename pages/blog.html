October 5, 2021 - Spooky Changes to the Website
Happy October! One of my favorite months for one of my favorite holidays. I love Halloween, especially when it comes to horror movies and scary decorations. I am not sure why, I just really like it.
Anyways, back to the portfolio. I titled it "Spooky" because today I had to initiate a massive change to the portfolio. Essentially, I had four projects to show in CSA, but I could only show three due to the size of the clock and holiday card. So, I created new pages. Easy enough, right? Not really. I had to change the menu for all the pages, create new CSS files, design all the new pages, link the pages, and a lot more. There were also some changes made to the menu that I wanted to make anyways, like a sideways arrow on the sub-menu. All those tasks are just about done, but I think I am gonna check in with Dr. R about it.
I knew I would have to have big maintenance for the site because this morning Dr. R said I would have to do something called refactoring. My GitHub site was not working because CSS files were not properly linked, so I would have to go back and change the files to call them correctly. Like before, though, it should be just about done.
