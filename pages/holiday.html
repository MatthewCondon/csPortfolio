Happy Holidays!

This is the project made as Holiday Card for 2021-2022. It was done using the Pair-Programming process. It was created in repl.it.

The hardest part of this project was designing the snowglobe. I had to get it to work in a specific location and now show snowflakes outside of the globe. This was done by just covering the outside of the globe with the same color as the background.

Working on the Countdown to 2022 was also difficult. However, this issue was solved with simple math. I took the time goal and subtracted the current time from it. It is sent to Central Standard Time.
