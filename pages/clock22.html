A Newer Clock

This is the clock project that we made before going on Winter Break. It was difficult in that it was an abstract clock. Given that I did a digital clock last year, I made an analog clock this year. I added a weather factor to it: type in a location and the project will respond with that location's temperature.

I really enjoyed the design process of this project. It was a lot of fun to move everything where I wanted it to go. I always liked making everything try to look as real as possible.

There were two very difficult pieces to this project. The first was using a JSIN file to grab the weather of a given location. The second difficult aspect was adding the numbers around the clocks. I tried using hard code on the main clock, but realized the process could be simpler. As such, I added a very specific for loop to add them.

Want to see the code? Here's the link: Clock Code

I hope this clock has everything you need!
